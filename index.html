About the Project

This website is a project for the course Web Applications Development 1 at Tafila Technical University. It is a fully functional web application designed to facilitate car booking services within Tafila Governorate. The platform allows users to easily browse available vehicles, check availability, and make reservations online.

The main goal of this project is to provide a modern, user-friendly, and efficient solution for car rental management in the region, making the process faster and more accessible for both customers and service providers.
